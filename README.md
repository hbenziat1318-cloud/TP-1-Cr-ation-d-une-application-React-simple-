
## TP 1 : Création d'une application React simple (Composants et Props)

##  Description

Ce projet est une **application React complète** développée dans le cadre d'un TP éducatif. L'objectif principal est de maîtriser les concepts fondamentaux de React à travers la création de composants réutilisables et la gestion des props.

##  Objectifs Pédagogiques

###  Concepts Maîtrisés
- **Composants Fonctionnels** - Création et utilisation
- **Props & Data Flow** - Transmission de données entre composants
- **JSX** - Syntaxe et expressions JavaScript
- **PropTypes** - Validation des types de données
- **Stylisation CSS** - Design moderne et responsive
- **Structure d'Application** - Organisation professionnelle

###  Compétences Développées
- Initialisation de projet avec Create React App
- Décomposition d'interface en composants
- Validation des données avec PropTypes
- Stylisation avancée avec CSS3
- Gestion des dépendances npm

##  Installation & Démarrage

### Prérequis
- **Node.js** version 14.0 ou supérieure
- **npm** version 6.0 ou supérieure

###  Installation

1. **Cloner le repository**
```bash
git clone <repository-url>
cd tp-react
```

2. **Installer les dépendances**
```bash
npm install
```

3. **Démarrer l'application**
```bash
npm start
```

###  Commandes Disponibles

| Commande | Description |
|----------|-------------|
| `npm start` | Lance le serveur de développement |
| `npm run build` | Crée une build de production |
| `npm test` | Lance la suite de tests |
| `npm run eject` | Éjecte la configuration (irréversible) |

###  Fonctionnalités Visibles
-  Messages de bienvenue personnalisés
-  Carte de profil avec image circulaire
